## Usage

### Search Reports
1. Type in the search box: document number, customer ID, or template ID
2. Results filter automatically as you type

### Filter by Date
1. Select start date to see reports from that date onwards
2. Select end date to see reports up to that date
3. Use both for a specific date range

### Clear Filters
- Click "Temizle" button to reset all filters
